A Git Together With Jon and Nate
===
Git is great, let's learn about it!

# What is Git?
- Distributed
- Open Source
- Version Control System

# Why should I use it?
- Code Snapshots
- History
- Collaboration
- Backups

# Vocabulary
- Version Control
- Repository
- Branch
- Commit

# Simple Git Commands
- `git help`
- `git config`
- `git init`
- `git status`
- `git add`
- `git commit -m "<commit_message>"`
- `git branch <new_branch>`
- `git checkout <branch>`
- `git merge <branch>`

# Remote Git Commands
- `git remote [add | remove] <remote_url>`
- `git push <remote> <branch>`
- `git pull <remote> <branch>`
